CREER L'ALGORITHME

- nombre de bonbons max qu'on peut acheter en fonction argent en possession et prix du bonbon

-  entrée : réel argent (argent dispo) = 43,26        réel prix (prix d'1 bonbon) = 2,11


===> Utiliser une BOUCLE


début acheterBonbons (43,26)

	Tant que 2,11 < 43,26
		2,11 <- 43,26 - 2,11
	Fin Tant que
	retourner entier

fin acheterBonbons
